🔐 Smart Login Hub

A modern and visually attractive Login Authentication Web Application built using HTML, CSS, and JavaScript, designed to deliver a secure and seamless user experience inspired by real-world platforms.

---

✨ Features

- User Registration and Login System
- Secure Authentication with Validation
- Protected Dashboard Access
- Logout Functionality
- Google Sign-In Option (Firebase Ready)

---

🎨 UI Highlights

- Split-screen modern layout
- Gradient design with clean typography
- Floating input labels
- Password show or hide toggle
- Smooth animations and hover effects
- Fully responsive across all devices

---

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage / Firebase

---

🚀 Project Objective

This project demonstrates practical implementation of authentication systems along with modern UI and responsive design principles.
